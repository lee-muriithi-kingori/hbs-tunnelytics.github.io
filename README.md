## Development

### Local Setup

```bash
# Clone repository
git clone https://github.com/lee-muriithi-kingori/hbs-tunnelytics.github.io.git

# Navigate and open
cd hbs-tunnelytics.github.io
open index.html
```

### Project Structure

```
hbs-tunnelytics.github.io/
├── .github/           # GitHub configuration
├── index.html         # Main website (single-page application)
└── README.md          # This file
```

> **Note:** The site is built as a single-page application with all content, styles, and scripts embedded in `index.html` for optimal loading performance on GitHub Pages.

## Deployment

The website is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

### Manual Deployment

1. Make changes to `index.html`
2. Commit and push:
   ```bash
   git add index.html
   git commit -m "Update website content"
   git push origin main
   ```
3. Changes will be live within a few minutes
